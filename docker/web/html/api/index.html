API Documentation

Welcome to the API server. Below are the available endpoints:

Accounts

  • GET /db/accounts - Retrieve all accounts
  • GET /db/accounts/:id - Retrieve a specific account
  • POST /db/accounts - Create a new account
  • PUT /db/accounts/:id - Update an existing account
  • DELETE /db/accounts/:id - Delete an account

Achievements

  • GET /db/achievements - Retrieve all achievements
  • GET /db/achievements/:id - Retrieve a specific achievement
  • POST /db/achievements - Create a new achievement
  • PUT /db/achievements/:id - Update an existing achievement
  • DELETE /db/achievements/:id - Delete an achievement

Game Scores

  • GET /db/game_scores - Retrieve all game scores
  • GET /db/game_scores/:id - Retrieve a specific game score
  • POST /db/game_scores - Create a new game score
  • PUT /db/game_scores/:id - Update an existing game score
  • DELETE /db/game_scores/:id - Delete a game score

Courses

  • GET /db/courses - Retrieve all courses
  • GET /db/courses/:id - Retrieve a specific course
  • POST /db/courses - Create a new course
  • PUT /db/courses/:id - Update an existing course
  • DELETE /db/courses/:id - Delete a course
